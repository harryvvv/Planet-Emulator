# Planet Emulator 🪐

2D симулятор Солнечной системы на Android.

## 📱 Возможности
- Загрузка планет из пресетов (все планеты Солнечной системы)
- Ручное добавление планет с заданием параметров
- Физическая модель гравитации
- Автоматический зум
- Отображение траекторий
- Лимит планет (настраиваемый)

## 🚀 Установка
1. Скачайте APK файл из репозитория
2. Установите на Android устройство (разрешите установку из неизвестных источников)
3. Запустите приложение
